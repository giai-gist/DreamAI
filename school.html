Experience 고급

1. 사업화 강의 (M) ① 창업 아이템이 있다면, 해당 아이템을 구현하기 위한 팀을 꾸리고, 창업 아이템이 없다면, 실제 수요가 있는 수요과제/경진대회 도전과제 중 목표 과제를 선정하여 창업팀을 구성합니다. ② 창업팀은 해당 과제를 분석하고, 제안서를 작성하며(사업계획서), IR 자료를 작성하여 발표합니다. ③ Pitching 단계에서는, 진행한 프로젝트 결과를 바탕으로 사업계획서를 보완하고, IR 자료를 작성하여 각종 공모전/경진대회/예비창업패키지에 도전할 수 있는 기반을 구축합니다. 특강) 현장에서 사업을 수행하는 창업기업, 스타트업 기업의 대표이사와의 화상회의를 통해 경험을 공유하고 네트워킹 하는 시간입니다. 2. 기술 강의 (T) ① 개인 별 로컬 장비 / 클라우드 환경에서 개발 환경을 구축합니다. ② MVP(Minimum Value Product) /Prototype 개발을 위한 AI를 개발하기 위해 데이터엔지니어링을 통해 필요 데이터 분석, 데이터셋 선정, 데이터 분석 기법을 정합니다. 데이터 전처리 위한 고급 기술로, 시계열데이터, 이미지 데이터, 음성 데이터 등 AI 개발을 위한 각종 데이터의 RAW 데이터를 가공하여 AI 개발 프레임웍에 쓸 수 있는 데이터셋을 개발하기 위한 과정입니다. ③ AI 프로토타입 개발 : Tensorflow, pyTorch 등 주류 프레임웍을 이용한 AI 개발 기법을 소개하며, 사업 아이템을 개발하기 위해 필요한 기법을 선정합니다. python 언어를 이용하여 AI 프로토타입을 개발합니다. ④ 서비스 개발: 앱/웹 개발을 위한 설계 및 기획 과정으로, AI 응용한 서비스를 수요처에 제공하기 위한 인터페이스인 앱/웹, 서비스 간 API 개발을 위한 과정입니다. ⑤ 실습 : 팀 별 프로젝트 진행을 강사진이 지원합니다. 3. 선행학습 및 개발환경 준비 선행학습 : - k-startup 의 '창업 에듀' 온라인 강의 중 '예비창업자' 과정 (https://www.k-startup.go.kr/edu/home/main/index) - python 기초 - github 사용 개발환경 준비: 개인용 장비(Windows/Ubuntu/Mac)에 개발 환경 설정 필수 - github desktop 설치, github 계정 생성 (https://desktop.github.com/ ) - visual studio code 또는, - python 개발 위한 pycharm, Java 위한 intelliJ, 또는 eclipse (https://www.jetbrains.com/pycharm/download/, https://www.jetbrains.com/idea/ ) - google cloud platform, kakao developer API, LINE api 등, 아이디어와 관련된 개발 플랫폼의 계정 생성 및 api 사용 key 발급 (https://cloud.google.com/?hl=ko, https://developers.kakao.com/, https://developers.line.biz/en/, https://developers.naver.com/main/ ) - microsoft 계정, teams 툴 가입 (https://www.microsoft.com/ko-kr/microsoft-365/microsoft-teams/group-chat-software) - zoom.us 가입 (https://www.zoom.us/) 4. 수강 이후 지원 ① 사업화 지원: 창업팀의 창업 과정, 창업팀의 경진대회, 정부지원 사업, 액셀러레이터의 공모전 등을 도전할 때 사업화 과정을 지원합니다. ② 기술 멘토링: AI 개발, 서비스 개발, 클라우드 서비스 운영 등 기술 지원 전반을 지원합니다. 대상: Expert 과정은 '고급' 과정으로, 초급 Experience 교육 범위의 내용인 '창업'에 대한 기본 지식과, 중급 Exercise 교육 범위인 소규모 프로젝트 수행을 경험을 보유한 수강생에게 추천합니다.
